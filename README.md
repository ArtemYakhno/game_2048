# 2048 Game

A browser-based implementation of the classic 2048 game built with vanilla JavaScript, HTML, and SCSS.

## About the Project

This project recreates the 2048 game for the web.
It includes full game logic, grid state management, and user interaction via keyboard and touch gestures on mobile devices.

The game features:
- Tile movement and merging logic,
- Keyboard controls (arrows),
- Swipe controls for mobile devices,
- Minimal animations for tile movement,
- Responsive layout for different screen sizes.

## Getting Started

Clone the repository:
git clone https://github.com/your-username/game_2048.git
cd game_2048

Install dependencies:
npm install
or
yarn install

Run the project locally:
npm start
or
yarn start
